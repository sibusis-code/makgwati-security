## Getting Started

### Local Development
1. Open the project folder in VS Code or your preferred editor
2. Right-click on `index.html` and select "Open with Live Server" (if you have the Live Server extension)
3. Or simply double-click `index.html` to open in your default browser

### Hosting Options

#### Option 1: GitHub Pages (Free)
1. Create a GitHub repository
2. Upload all files to the repository
3. Go to repository Settings > Pages
4. Select source branch (usually `main`)
5. Your site will be live at `https://yourusername.github.io/repository-name`

#### Option 2: Netlify (Free)
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop the project folder to deploy
3. Get instant live URL

#### Option 3: Traditional Web Hosting
- Upload files to your web hosting provider's public folder (usually `public_html` or `www`)

## Customization Guide

### Colors
The website uses a professional color scheme suitable for the security industry:
- **Primary Blue**: #1a365d (Dark blue for headers and text)
- **Secondary Blue**: #3182ce (Medium blue for accents)
- **Orange Gradient**: #FF6B35 to #F7931E (For highlights and buttons)
- **Gray**: #64748b (For secondary text)

To change colors, edit the CSS variables or search and replace color values in `styles.css`.

### Content Updates
- **Business Information**: Update contact details in the Contact section of `index.html`
- **Pricing**: Modify prices in the Services section
- **Services**: Add or remove training programs in the services grid
- **About Section**: Update company information and statistics

### Adding New Sections
1. Add HTML structure to `index.html`
2. Add corresponding CSS styles to `styles.css`
3. Update navigation menu if needed
4. Add any interactive functionality to `script.js`
